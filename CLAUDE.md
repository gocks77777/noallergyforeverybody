## 주요 API 엔드포인트 (목표)

- `POST /predict` — 이미지 → 음식명 + 알레르기 분석
- `GET /barcode/{code}` — 바코드 → 성분 + 알레르기
- `GET /restaurants` — 주변 식당 지도 (OA-16094)
- `GET /hotspots` — 외국인 밀집 지역 경보 (OA-14993)

## 추론 흐름

```
이미지 입력
  → ViT 임베딩 → FAISS 검색 → 음식명 (top-3)
  → 공공급식 식재료 매핑 (OA-20918)
  → Claude 3.5 Sonnet: 알레르기 위험 분석
  → SQLite 캐시 저장 (동일 쿼리 재요청 시 $0)
```
